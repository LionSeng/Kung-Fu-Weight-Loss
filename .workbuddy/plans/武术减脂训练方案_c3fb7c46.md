## 产品概述

创建一个交互式HTML训练看板，将武术基础训练与科学减脂融合，帮助有一定运动基础的用户在8周内实现燃脂减重与体魄锤炼的双重目标。

## 核心功能

- **8周进阶训练计划**：每周5天训练+2天恢复，从基础适应性训练逐步过渡到高强度搏击体能
- **每日训练详情**：每次60-90分钟，包含热身(10min) → 技术训练(25-35min) → 体能强化(20-30min) → 拉伸(10min) 四个阶段
- **武术风格推荐**：以搏击类（散打/拳击）为核心，辅以体能训练，HIIT属性强、燃脂效率高
- **训练看板**：按周/日维度展示训练内容，支持进度追踪打卡
- **饮食建议**：配合训练的营养指导，高蛋白适量碳水的减脂饮食方案
- **训练原则与注意事项**：包含热身、恢复、受伤预防等关键指导

## 用户画像

- 偶尔锻炼，有一定运动基础（跑步、健身等）
- 每周训练天数灵活，方案建议5天/周
- 需要移动端友好的可视化页面

## 技术栈

- **单文件HTML**：自包含，无外部依赖（内联CSS+JS），直接浏览器打开即可使用
- **数据存储**：LocalStorage 持久化训练打卡记录
- **响应式设计**：CSS媒体查询适配桌面端和移动端

## 实现方案

采用单HTML文件方案，内嵌所有样式和逻辑。使用CSS Grid/Flexbox构建响应式布局，原生JavaScript管理交互状态。训练计划数据以JSON数组内嵌在JS中，按8周×7天结构组织。LocalStorage存储打卡状态，支持进度可视化。通过CSS变量定义主题色系，渐变配色营造运动氛围。

## 架构设计

- 数据层：8周训练计划JSON数据（周次/天次/训练阶段/具体动作/组数/时长）
- 视图层：顶部Tab切换（计划总览/本周训练/饮食建议/训练须知），卡片式训练内容展示
- 交互层：日期选择 → 展示当日训练 → 完成打卡 → 进度条更新 → LocalStorage持久化

## 设计风格

采用 **Cyberpunk Neon UI** 混合 **运动科技风**，深色背景搭配活力霓虹色，营造热血运动氛围。使用深灰/黑色为底，搭配荧光绿、电光蓝、橙红作为功能色，呼应武术的力量感与减脂的活力感。

## 页面规划

- **顶部导航栏**：应用标题 + 4个Tab（训练计划/饮食方案/训练须知/进度统计）
- **训练计划页（主页面）**：
- Block 1：8周进度总览条（横向进度条，标注当前周次）
- Block 2：周次选择器（横向可滑动周次卡片）
- Block 3：本周训练日历（7天卡片，训练日/恢复日颜色区分，打卡状态标记）
- Block 4：当日训练详情（四个阶段卡片：热身/技术/体能/拉伸，每个卡片含动作名+组数+时长）
- Block 5：打卡按钮（大按钮，点击后动画确认）
- **饮食方案页**：减脂期饮食原则 + 每日热量参考 + 推荐食谱
- **训练须知页**：热身要点/受伤预防/恢复建议/装备推荐
- **进度统计页**：已完成天数/连续打卡/燃脂预估

## 交互设计

- Tab切换带底部滑动指示器动画
- 周次选择平滑过渡
- 打卡按钮点击后弹出成功动画（勾号+光效）
- 训练卡片hover微浮起效果
- 移动端友好，单列布局适配手机
